# Daily Sync

일일 동기화를 수행합니다. 매일 작업 시작 전 실행을 권장합니다.

## 사용법
```
/daily-sync
```

## 실행 스크립트

### 1. Git 상태 확인
```bash
echo "=== Git 상태 ==="
git status -s
git branch -vv
git log --oneline -5
```

### 2. 빌드 상태 확인
```bash
echo "=== 백엔드 빌드 ==="
cd backend
./gradlew build -x test --quiet

echo "=== 프론트엔드 빌드 ==="
cd frontend
npm run build --silent
```

### 3. 테스트 상태 확인
```bash
echo "=== 백엔드 테스트 ==="
./gradlew test --quiet

echo "=== 프론트엔드 테스트 ==="
npm run test --silent
```

### 4. 서버 상태 확인
```bash
echo "=== 서버 상태 ==="
curl -s http://localhost:8093/actuator/health | jq '.status'
curl -s http://localhost:8110 > /dev/null && echo "Frontend: OK" || echo "Frontend: DOWN"
```
