## 새 문서 작성 가이드

[`_template.md`](./_template.md) 템플릿을 복사하여 새 문서를 작성하세요.

### 필수 섹션

1. **기본 정보** - 저장소, 제작자, 라이선스, 분류
2. **개요** - 무엇인지, 어떤 문제 해결
3. **주요 기능** - 핵심 기능 목록
4. **설치 방법** - 설치 명령어
5. **사용 예시** - 실제 사용 예시
6. **장단점** - 장점, 주의사항
7. **관련 리소스** - 관련 링크
